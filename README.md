# OpusPlayer

OpusPlayer is a simple cross-platform console/terminal opus encoded audio file player.

### Dependencies

SDL2 opusfile libogg opus

### Usage

```
main
INFO: Usage: main [OPTIONS] filename
INFO: Options:
INFO: -volume (1-100)
```
